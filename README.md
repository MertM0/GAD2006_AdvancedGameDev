# GAD2006 - Advanced Game Development

![Unreal Engine](https://img.shields.io/badge/Unreal_Engine-5.x-black?logo=unrealengine)
![C++](https://img.shields.io/badge/C++-Used-blue?logo=c%2B%2B)

This repository contains the labwork projects developed with **Unreal Engine 5** and **C++** for the **GAD2006 Advanced Game Development** course. Each labwork consists of a separate Unreal Engine project in its own folder.

## Labworks

### [Labwork 1](./Labwork1/) - 26.03.2026
Custom BaseCharacter and AvatarCharacter.<br>Includes Character Movement, Camera Control, Input Bindings, and a Health & Stamina System.

### [Labwork 2](./Labwork2/) - 16.04.2026
Command-Based Game Template.<br>Includes GameGrid generation, GameSlot interactions, and a Command Pattern for unit movement with a Undo/Revert system.
