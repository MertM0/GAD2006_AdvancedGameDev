# GAD2006 - Advanced Game Development

![Unreal Engine](https://img.shields.io/badge/Unreal_Engine-5.x-black?logo=unrealengine)
![C++](https://img.shields.io/badge/C++-Used-blue?logo=c%2B%2B)

This project is being developed with **Unreal Engine 5** and **C++** for the **GAD2006 Advanced Game Development** course.

## Features
- **Unreal Engine 5 and C++ Architecture**
- **Custom Base Character:** Includes foundational parameters such as `Health` and basic attributes.
- **Avatar Character:** Extends the base character to include features like `Sprinting`, `Stamina Management`, and `Third-Person Camera Controls`.


## Controls
| Action | Key / Input |
| :--- | :--- |
| **Move** | `W` `A` `S` `D` |
| **Look** | `Mouse` |
| **Jump** | `Space` |
| **Sprint** | `Left Shift` |
